# tiket-pembaris-backend

Backend untuk aplikasi tiketbaris.

## Local development

- Copy `.env.example` to `.env` and adjust values.

## OpenAI integration

- Set `OPENAI_API_KEY` in your `.env` to enable AI features.
- Set `OPENAI_DEFAULT_MODEL=gpt-5-mini` to make GPT-5 mini the default model used by the helper.

Example .env entries:

OPENAI_API_KEY=sk-...
OPENAI_DEFAULT_MODEL=gpt-5-mini
